DrugAI-GAN.py

Experiments with GAN for drug like molecule generation.

If you are visiting here for the first time then I would recommend you to my earlier project DrugAI before reading any further. The main aim of this project is to carry out some experiments with Generative adversarial networks (GAN) for sequence generation. GAN's are discoverd by Ian Goodfellow in 2014 for image generation. It has revolutinized the way images are generated using machines and there are currently many research group involved in this algorith. There are various types of GANs such as stackGAN, cycleGAN, inforGAN, dcGAN,pix2pix etc. I would also recommend you to go through those projects and see for yourself the sureal images created by GANs.


In my previous DrugAI sub project I have used a LSTM model for generating drug like molecule. I want to see weather these kind of molecules can also be generated using GANs. While searching for application for GAN in text generation; I came across a reddit post by Ian Goodfellow himself. According to him and an another research paper, its very hard to teach GAN in a discrete dataset. As sequence generation involves discrete data it would be very hard to generate any meaning ful sequence from it. This is just an experiment to see for myself; what happens if GAN is used for sequence generation.

Most of the code for this project I salvaged from my earlier LSTM model and added an additional GAN model to it. The GAN model was prepared using the code and recommendations from web articles freely available in the web and I am in no way guaranteing its accuracy. Instead of using a gaussian noise as input, I chose my earlier LSTM model x values as the input.

Because I have a very old system, training most of the advanced algorithm is not possible. I recently came across floydhub.com which has a free tier for training deep learning model using either gpu or cpu. I have heard about AWS cloud system for deep learning but it requires a submission of credit card or bank account details which I am not confortable with. This is the first time I am using a cloud server for training deep learning projects and so far I am very much impressed by floydhub and its support. The web interface is very simple and evenafter being a first time user I can easily deploy my project in floydhub. There were initially some hiccups but their support was extraordinary and fast.

I deployed my GAN project this way in floydhub

  • Download DrugAI-GAN.py and stahl.csv dataset from github

  • Copy and save those files inside a new folder with name DrugAI

  • Install floydhub command line tool $ pip install -U floyd-cli

  • Login to floy hub $ floyd login

  • From shell CD and go to DrugAI folder

  • Initilaze a new project $ floyd init DrugAI-GAN

  • Run this code $ floyd run "python DrugAI-GAN.py"

Even with a cloud service like floydhub, it was taking me more than a day to train with complete dataset. In free tier a project will automatically time out after 24 hours in floydhub, which is a limitation for this service. Instead of using complete dataset(>300 rows) I used only 30 rows. This allowed me to train my model within 2 or 3 hrs and see the results immeditely.
