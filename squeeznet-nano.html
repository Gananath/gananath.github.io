Squeeznet-nano

Image classifiction of nano materials from SEM and TEM images using squeeznet

This project is my first endevour to image classification and transfer learning. I have used squeezenet for this learning with a slight modifications. The last softmax layer of squeeznet is removed and additional dense and droupout layers were added to new modified architecture. SqueezeNet achieves AlexNet-level accuracy on ImageNet with 50x fewer parameters. That is squeezent learns very fast and resultend model size also tends to be very small. The original alexnet and squeeznet is >250 MB and 5 MB respectevily whereas my modified model is approximately 15MB in size.

Initially I have used RELU as the activation funcion for all the new layers except the last layer which I kept as softmax. Somehow addition of the RELU activation fuction to the new layers overwhelmed my neural netowork and it dosent learned anything and showed an error in loss as NaN. This error I think is mainly caused by the range of RELU output which is [0,+infinity). Then I changed starting activation function to TANH whose range is between (-1, 1) and now it trained without any error.

Predicting targets

  • Nanotubes

  • Nanoparticles

  • None or Random

Caution: As this is first time I am implementing a image classification and transfer learning; I am not sure about the accuracy of the "transfered" model. I also trained the model for very less computation and with a small dataset because of my lack of infrastruture. So I will advice you to take the code and results with a grain of salt.
